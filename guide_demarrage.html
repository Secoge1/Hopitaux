Guide de Démarrage Rapide

3 Étapes simples pour corriger les anomalies
1

Lancer les tests automatiques

Vérifiez l'état actuel de votre système en exécutant les tests automatiques.

Ce que fait ce test :
  • Vérifie la structure de la base de données
  • Contrôle les corrections appliquées au code
  • Affiche les statistiques actuelles
  • Identifie les problèmes restants
Exécuter les tests maintenant
2

Corriger la base de données

Si les tests révèlent des problèmes, exécutez le script de correction automatique.

Ce que fait ce script :
  • Ajoute la valeur 'supprime' aux ENUM si absente
  • Crée les colonnes date_suppression si absentes
  • Affiche un rapport détaillé des modifications
  • Invalide le cache pour rafraîchir les compteurs
Sécurité : Ce script ne supprime aucune donnée, il modifie uniquement la structure de la base.
Corriger la base maintenant
3

Vérifier et tester

Effectuez les tests manuels pour vous assurer que tout fonctionne correctement.

Tests à effectuer :
Test 1 : Supprimer un patient et vérifier que le compteur diminue
Test 2 : Supprimer un médecin et vérifier que le compteur diminue
Test 3 : Supprimer une consultation et vérifier que le compteur diminue
Test 4 : Restaurer un patient et vérifier que le compteur augmente
