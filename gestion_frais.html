Qu'est-ce qu'un soin ?

Les soins sont les actes médicaux que vous pouvez ajouter lors d'une consultation : pansements, injections, sutures, etc. Chaque soin a un nom, un type, un prix et une durée.

📋 Types de soins disponibles :
Pansements
Injections
Examens
Soins intensifs
Perfusions
Petite chirurgie

💰 Tarifs de Consultation

Qu'est-ce qu'un tarif de consultation ?

Les tarifs de consultation définissent le prix de base d'une consultation selon son type (normale, urgence, domicile) et la spécialité du médecin (généraliste, cardiologue, pédiatre, etc.).

📊 Types de consultation :
  • Consultation normale - Visite planifiée
  • Consultation d'urgence - Intervention rapide
  • Consultation à domicile - Visite du médecin
  • Consultation de suivi - Contrôle régulier
🩺 Spécialités médicales :
  • Cardiologie
  • Pédiatrie
  • Neurologie
  • Médecine générale

❓ Comment utiliser

Pour ajouter un soin :
  1. Cliquez sur "Gérer le catalogue des soins"
  2. Cliquez sur "Ajouter un soin"
  3. Remplissez :
    • Nom du soin (ex: "Pansement simple")
    • Type de soin (ex: "Pansement")
    • Prix (ex: 2500 FCFA)
    • Durée estimée (ex: 15 minutes)
    • Description (optionnel)
  4. Enregistrez
Pour modifier un tarif :
  1. Cliquez sur "Gérer les tarifs de consultation"
  2. Trouvez le tarif à modifier
  3. Cliquez sur "Modifier"
  4. Changez :
    • Le prix
    • Le type de consultation
    • La spécialité
    • La description
  5. Enregistrez les modifications
💡 Conseils pratiques :
Nomenclature claire :
Utilisez des noms précis et compréhensibles pour vos soins (ex: "Pansement simple" plutôt que "Pansement")
Prix réalistes :
Basez-vous sur les tarifs du marché et ajustez selon votre structure
Révision régulière :
Mettez à jour vos tarifs au moins une fois par an
